# Log Monitor Frontend

Приложение для сбора и визуализации ошибок и логов из браузерных расширений.

## Стек

- **Next.js 16** — фреймворк
- **React 19** — UI
- **TypeScript** — типизация
- **Redux Toolkit** — глобальное состояние
- **TanStack Query** — кэширование API запросов
- **Ant Design** — UI компоненты
- **Chart.js / @ant-design/charts** — графики
- **Axios** — HTTP клиент
- **Zod** — валидация данных

## Быстрый старт

```bash
npm install
npm run dev
```

Приложение доступно по адресу `http://localhost:3000`
